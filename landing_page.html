WeatherPy

***Don't yet understand how to control text colors, so far can't escape the blue without the dumb trick hidden in the next line.***

For this not very scientific study of global weather patterns, cities were first selected from the Python library Citipy using randomly generated pairs of lat/long coordinates, with the goal of collecting a list of at least 500 cities for which complete weather data were available. Weather data was acquired from the OpenWeatherMap API. Note that data represents only a single timepoint on a single day; it will have been daytime in some places and nighttime in others. Of 628 cities in the original list, 582 were retained following inspection for missing or unreliable-seeming values.

The results were then plotted to look for latitudinal trends in maximum temperature, relative humidity, cloudiness, and wind speed. From the thumbnails at the left, it is already clear that only temperature varied in a relatively linear fashion from the equator to the poles. A few more details are discussed on the individual pages.
